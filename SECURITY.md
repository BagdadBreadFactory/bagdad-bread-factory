# 🛡️ Security Policy

We take the security of **Bagdad Bread Factory & Sweets** seriously. If you believe you have discovered a security vulnerability, we appreciate your help in disclosing it to us in a responsible manner.

---

## 📋 Supported Versions

Only the latest version on the main branch is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |
| < 1.0   | :x:                |

---

## 🚨 Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a vulnerability, please report it privately to our team so we can address it as quickly as possible.

### Contact Info
Send an email detailing your findings to:
* **Primary:** `bagdadbreadfactoryofficial@gmail.com`
* **Secondary:** `info.bagdadfactory@gmail.com`

---

## 📝 What to Include in Your Report

To help us understand and resolve the issue quickly, please include as much detail as possible:

1. **Type of Vulnerability:** (e.g., XSS, broken link hijacking, header misconfiguration, data exposure)
2. **Steps to Reproduce:** Clear, step-by-step instructions or a minimal code snippet.
3. **Impact:** Potential risks and what an attacker could achieve.
4. **Proof of Concept (PoC):** Screenshots, requests/responses, or video recordings (if available).
5. **Mitigation Ideas:** (Optional) Any suggestions or code fixes you might have.

---

## ⏱️ Response Policy & Timeline

When you report a vulnerability to us:

* **Acknowledgment:** We will acknowledge receipt of your report within **24–48 hours**.
* **Assessment:** We will review and attempt to validate the issue within **3 to 5 business days**.
* **Resolution:** If verified, we will prioritize a fix and notify you once the deployment is complete.

---

## 🤝 Responsible Disclosure Guidelines

To protect our users and project integrity:

* Give us reasonable time to investigate and address the vulnerability before making any information public.
* Do not access, modify, or destroy user data or system integrity.
* Act in good faith to avoid privacy violations, disruption to our services, and destruction of data.

Thank you for helping us keep **Bagdad Bread Factory & Sweets** safe and secure! 🥖✨
