# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

Please do not create a public GitHub issue for security-related problems.

Instead, contact us using the following email:

**Email:** bagdadbreadfactoryofficial@gmail.com

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- Any screenshots or proof of concept (if available)

We will review your report as soon as possible and work to resolve the issue.

Thank you for helping keep this project secure.
